# Wrap up Type I and Type II Errors

It is important to remember to take into consideration the possibility of the occurrence of Type I or Type II error, when drawing conclusions from hypothesis tests. Thus:

- Whenever there is a failure to reject the null hypothesis, it is possible that a
				Type II error has occurred. Thus, it is concluded from the results of the study,
				that there are no significant differences, even though, in reality, there are
				significant differences.
- Whenever the null hypothesis is rejected, it is possible that a Type I error has
				been committed. That is, it is concluded from the results of the study, that there
				are significant differences, when, in reality, there are no differences.

However, it is not possible to know when either Type I or Type II error has occurred.
